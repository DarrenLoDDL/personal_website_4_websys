Experiences with Computing in High School

For Year 9 and 10, I took a subject called Information Software and Technology (IST). We programmed in Scratch as an introduction to coding, and moved onto Python as we got better. The hardest thing that I programmed was a quiz "game" that simply had users select a, b, c, or d. After finishing Year 10, I haven not touched anything programming related until my first year of university.

Experiences with Computing in university

I obviously did a LOT more programming, especially when my entire degree revovles around it. In 1st year, I took the course Programming Fundamentals, where we learnt to code in Java, and Introduction to Mathematical Modelling and Analysis, where we learnt to use Python to graph functions, and solve other mathematical problems. Moving onto 2nd year, which is this year, I have learnt how to create an application using JavaFX. For a project that I undertook in Computer Science Studio 1, I learnt how to create Convolutional Neural Networks on Python, using the Keras API, to analyse and classify medical images. Right now, I am learning about the theory of quantum computation, and using Qiskit to simulate quantum algorithms. I am also learning how create a website for Web Systems :D.
